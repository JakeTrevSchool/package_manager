About Package Manager & how you can contact us

Package Manager has been developed to help users create, share, and leave feedback on each other's packages. As a result, people have the ability to download tools that will help them in their personal projects as well as publish their existing projects in order to obtain feedback on how to improve them.
Package manager, just like any other package here, is a project that may not be fault-free. If you come across any issues while using our site or have feedback on how we can improve your experience by adding new features, please do not hesitate to contact us.
